This project has been done in the second year of college.


It aims to familiarize us with the use of streams.

In the file Activities.txt, there are a lot of activities printed in the format <BeginDate> <EndDate> <Activity Name> and the task was filter those activities in order to print some requested date,
like:
    -Number of Times each activity has been done overall(Cerinta 2)
    -Number of Times each activity has been done per day(Cerinta 3)
    -etc.
